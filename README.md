# DIET

Єдине джерело правди для журналу ваги, харчування і тренувань.

## Принципи

- Не фантазувати дані.
- Якщо вага або грамовка не вказана — не вигадувати.
- Всі оцінки калорій/БЖВ мають бути позначені як estimate.
- Сирі записи з чату зберігати окремо від розрахунків.
- Формат звітів не міняти без явної команди.
- Не створювати 300+ денних файлів: один тиждень = один файл.

## Структура

```text
profile.json
weight.csv
weeks/
  2026-W25.json
  2026-W26.json
  2026-W27.json
reports/
charts/
scripts/
```

## Поточна фаза

- Стартова вага фази: 98.0 кг
- Ціль: точний контроль харчування, ваги, тренувань і плато
- Поточна проблема: вага стоїть/плато, потрібен контроль без зайвих порад

## Правило запису

Кожен новий запис має одразу потрапляти в Git.

Основний формат даних: `weeks/YYYY-Wxx.json`, де один файл містить усі записи за тиждень.

`weight.csv` залишається окремим індексом ваги для швидких графіків.
