# DIET

Єдине джерело правди для журналу ваги, харчування і тренувань.

## Принципи

- Не фантазувати дані.
- Якщо вага або грамовка не вказана — не вигадувати.
- Всі оцінки калорій/БЖВ мають бути позначені як estimate.
- Сирі записи з чату зберігати окремо від розрахунків.
- Формат звітів не міняти без явної команди.

## Структура

```text
profile.json
weight.csv
data/raw/import_2026-06-15_to_2026-06-27.json
reports/
charts/
scripts/
```

## Поточна фаза

- Стартова вага фази: 98.0 кг
- Ціль: жорсткий контроль харчування, ваги, тренувань і плато
- Поточна проблема: вага стоїть/плато, потрібен контроль, а не мотиваційні фрази

## Правило запису

Кожен новий день має потрапляти в Git як структурований запис: дата, вага, прийоми їжі, тренування, коментарі, рівень точності даних.
